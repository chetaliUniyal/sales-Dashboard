This project is an interactive E-Commerce sales Dashboard developed using Power Bi to analyze and visualize online sales data. The Dashboard provides valuable insights into sales performance, profit trends, customer behaviour, and product categgories.

KEY FEATURES
1. Analyzed sales, profit, quantity sold, and order trends
2. Created interactive charts and visualizations for better decision-making
3. Identified top-performing products and high-revenue regions
4. Tracked monthly sales and profit performance
5. Used data cleaning and transformation techniques to improve data quality

1.Power BI
2.Microsoft excel
3. Data visua;ization
4.Data cleaning & Transformation
5. Analytics
